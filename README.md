# imba-rotn

ROT-N app in Imba. [You can see in in action here](https://taw.github.io/imba-rotn).

## Install dependencies

```
npm install
```

### Run in development mode

```
npm run dev
```

### Build

```
npm run build
```

### Run tests interactively

Start server then run:

```
$(npm bin)/cypress open
```
